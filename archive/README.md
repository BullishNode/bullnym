# Archive

This directory contains historical evidence. It is not the current product or
operational contract.

- [Plans](plans/README.md): implemented, superseded, or abandoned proposals.
- [Research](research/README.md): time-bound investigations and vendor notes.
- [Testing evidence](testing-evidence/README.md): historical audits and coverage notes.

Archived files must state their lifecycle at the top. Do not revive an archived
design by editing it in place; create an RFC and link the historical source.
